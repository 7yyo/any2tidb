方案对比

A

前后端一体

Spring Boot 同时 serve 前端静态文件 + REST API。一个 JAR 部署。

Pros

  • 部署简单，一个 JAR
  • 无跨域问题

Cons

  • 前后端耦合
  • 前端开发需 Java 环境
B

前后端分离（推荐）

React 独立开发，生产 nginx 反代。开发体验好。

Pros

  • 独立开发 Vite HMR
  • nginx 反代干净

Cons

  • 需额外 nginx 配置
